Bienvenidos a mi Portafolio de Bitlab

En este pequeño portafolio encontrarás parte de las tareas que realice en el curso de Introducción a la Programación de Bitlab, además de las herramientas en las cuales se desarrollaron, entre otras cosas.


Proyectos


Formulario no funcional de inscripciones a la NASA

Descripción: Para esta tarea se pidió un pequeño formulario de inscripciones para ser parte de la NASA, las herramientas a utilizar para esta tarea fueron: HTML5 y CSS.

To Do List

Descripción: esta tarea se requería de un archivo html, el cual permitiera ingresar tareas pendientes por realizar, y que al finalizar con estas pudieran ser eliminadas. Herramientas utilizadas: HTML5, CSS y JavaScript.

Calculadora de propinas

Descripción: en esta tarea se pidió realizar una Calculadora de propinas, en la cual se pedia el valor a cancelar de la cuenta, el porcentaje de propina a pagar, y la cantidad de personas que cancelarán la cuenta. Herramientas utilizadas: HTML5, CSS y JavaScript.

Página web de un restaurante (no funcional)

Descripción: En esta tarea se pidió realizar una pequeña página web de cualquier restaurante, con la dificultad que debía tener la funcionalidad de envíar un mensaje de reservación a cierto correo electrónico. Herramientas utilizadas: HTML5, CSS, JavaScript, PHP y Bootstrap.
